# Referência de regiões e localidades

Este documento registra a lista bruta enviada pelo usuário para orientar conteúdo, mocks e refinamentos futuros da seção de regiões atendidas.

## Uso atual no roadmap

- Na Onda 1, a seção de regiões deve usar um mock/principais localidades, sem tentar publicar a lista completa como definitiva.
- A lista completa precisa de revisão, agrupamento, normalização ortográfica e priorização antes de virar conteúdo final.
- A disponibilidade exata continua dependendo da coordenada e da API de viabilidade.
- Não usar esta lista para desenhar cobertura técnica, rotas, torres, CTOs ou tecnologia provável.
