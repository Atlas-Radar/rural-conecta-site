# Design System

DRAFT inicial.

## Personalidade

Rural, premium, tecnológica, humana, regional e confiável.

## Tokens de forma e ritmo

- Container máximo: `72rem`
- Padding lateral mobile: `1rem`
- Padding lateral amplo: `1.5rem`
- Padding de seção mobile: `4rem`
- Padding de seção desktop: `6rem`
- Radius de card: `1.25rem`
- Radius de card destacado: `1.5rem`
- Radius de botão/chip: `999px`
- Altura mínima de toque: `3rem`
- Botão grande preparado: `3.5rem`
- Sombras: `soft`, `elevated` e `glow` controlado por tokens CSS.
- Transições: apenas cor, borda, sombra e transform em `160ms`.

## Direção Etapa 0

As referências visuais aprovadas foram traduzidas em tokens para uma base
dark premium rural-tech: fundo quase preto com subtom verde, cards escuros
translúcidos, bordas finas, acento verde vivo com moderação, foco visível e
uma seção clara futura para quebrar o ritmo da landing. A tipografia permanece
em fonte de sistema para máxima performance.

## Regras

- Mobile-first.
- Fonte de sistema ou uma fonte local leve.
- CTA verde com moderação.
- Não usar texto dentro de imagens.
- Não usar vídeo, partículas, 3D pesado ou blur em excesso.
- Conteúdo comercial não confirmado deve ser placeholder claro.
