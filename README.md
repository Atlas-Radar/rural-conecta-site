# Rural Conecta Site

Fundação do site institucional e landing page da Rural Conecta.

## Objetivo
Construir uma landing page estática, moderna, mobile-first e leve para uma provedora de internet rural, com consulta simples de disponibilidade e atendimento final por WhatsApp.

## Status
Fundação local inicial. A landing completa, Google Maps, API real, deploy e GitHub remoto ainda não foram implementados.

## Stack planejada
- Astro static-first
- TypeScript strict
- CSS nativo
- TypeScript vanilla no navegador
- Vitest para lógica crítica
- Playwright somente quando houver fluxo E2E real
