# Design System

## Status

DRAFT inicial.

## Personalidade

Rural, premium, tecnológica, humana, regional e confiável.

## Tokens provisórios

- Background 950: #020B0C
- Background 900: #061514

- Surface 700: #0D2A21

- Text primary: #F5F7F3
- Text muted: #B5C0B8

## Regras

- Mobile-first.
- Fonte de sistema ou uma fonte local leve.
- CTA verde com moderação.
- Não usar texto dentro de imagens.
- Não usar vídeo, partículas, 3D pesado ou blur em excesso.
- Conteúdo comercial não confirmado deve ser placeholder claro.
